# 角色 Canon (CANON_CHARACTER_XueZhiYing)

> **角色**: 薛知盈 (Xue Zhiying)
> **身份**: 小说主角，出身不凡但身陷京城纷争的清冷女子
> **级别**: GRADE S (8K Ready)

## 1. 物理特征 (Physical Traits)

- **气质**: 清冷、内敛、眼底带有一丝倔强。
- **面部特征**:
  - 标准中式鹅蛋脸。
  - 眉形微挑，色深且淡（非浓眉）。
  - 双眼皮极窄，眼尾微扬，带有一点“凤眼”神韵。
  - 皮肤：冷白皮，质感通透。
- **发型**:
  - 典型的京城世家女子发髻，不追求繁复，但需有极细腻的垂丝感。
  - 发饰：一支素净的白玉簪，发丝质感需达到 8K 级可见根根分明。

## 2. 服饰结构 (Clothing Structure)

- **基准**:
  - 内部为淡青色立领中衣。
  - 外部为重叠丝缎结构的对襟长衫，材质需体现出轻盈且有骨架感。
  - 样品披风：雪青色狐皮滚边披风。
- **细节要求**: 刺绣花纹需为物理凹凸感，非平涂贴图。

## 3. 禁止项 (Forbidden Traits) - 审美审计红线

- ❌ 严禁使用网红大眼妆、欧式大双。
- ❌ 严禁发髻出现非重力下垂的乱发。
- ❌ 严禁眼神出现任何呆滞或无神的 Staring。
- ❌ 严禁服饰材质出现任何“塑料感”高光。

## 4. 视角一致性

任何侧面、背面视图必须符合正面设定的头骨结构及其发髻物理结构。
