# 环境 Canon (CANON_ENV_InnPrivateRoom)

> **场景**: 客栈雅间 (Inn Private Room)
> **级别**: GRADE S (8K Ready)

## 1. 地理架构 (Spatial Architecture)

- **布局**:
  - 窗户位于画面左侧，面朝街道（可见落花）。
  - 书案房位于室内进深处。
  - 回廊连接外间，由屏风或珠帘隔断。
- **材质**:
  - 地板：暗红色漆木，具有自然的磨损纹理。
  - 墙面：宣纸质感的屏风结合木质框架。
  - 窗框：刻有精细中式花纹的实木。

## 2. 光照氛围 (Lighting & Atmosphere)

- **时间段**: 午后 (Afternoon)。
- **光线**:
  - 主光：来自左侧窗户的散射日光，带有落花的影子映射。
  - 补光：室内柔和的暖光。
  - 阴影：角色由左向右投射淡雅的逻辑阴影。
- **氛围**: 尘埃颗粒在光柱中可辨识（8K 细节），宁静中带有压抑感。

## 3. 核心道具 (Core Props)

- **窗台**: 需有真实的木质纹理与尘土感。
- **披风**: 雪青色，丝织质感。
- **书案**: 笔墨纸砚布局规整。
- **燕窝盏**: 王嬷嬷手中，白玉瓷质。

## 4. 审美审计红线

- ❌ 严禁出现任何违背地理架构的穿模。
- ❌ 严禁使用饱和度过高的“廉价”色彩组合。
- ❌ 严禁光线出现无源高光（必须遵循物理光源）。
