## 四、关键阻塞项分析

### 4.1 P0 阻塞项（必须解决）

#### 1. Worker 运行环境验证 ⚠️
- **问题**: Worker 需要 FFmpeg 才能处理 VIDEO_RENDER Job
- **影响**: 视频生成功能无法工作
- **验证方法**:
  ```bash
  # 在 Worker 环境执行
  which ffmpeg
  ffmpeg -version
  ```
- **修复建议**: 
  - 如果缺失，在 Worker Docker 镜像中安装 FFmpeg
  - 或使用包含 FFmpeg 的基础镜像

#### 2. Worker API 连接验证 ⚠️
- **问题**: Worker 需要能连接 API 并领取 Job
- **影响**: Worker 无法处理任何 Job
- **验证方法**:
  ```bash
  # 启动 Worker，检查日志
  pnpm --filter workers dev
  # 应该看到: Worker registered successfully
  ```
- **修复建议**: 
  - 确认 `WORKER_API_KEY` 和 `WORKER_API_SECRET` 配置正确
  - 确认 `API_BASE_URL` 可访问

#### 3. 视频文件访问验证 ⚠️
- **问题**: 生成的视频文件需要能通过 API 访问
- **影响**: 用户无法下载/预览视频
- **验证方法**:
  ```bash
  # 运行 video E2E 测试
  bash tools/smoke/run_video_e2e.sh
  # 检查: HTTP_CODE == 200
  ```
- **修复建议**: 
  - 确认 `GET /api/storage/:key` 端点正常工作
  - 确认存储路径配置正确
