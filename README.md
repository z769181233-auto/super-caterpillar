## 文档 / Documentation

### Stage3 总览（多引擎与统一引擎信息架构）

- **文档位置**：`docs/STAGE3_OVERVIEW.md`
- **文档版本**：v2.0（核心能力 100%，S3-C Phase 2/3 为可选增强）
- **文档名称**：《STAGE3 总览文档｜正式版（中文）》

Stage3 在 Stage2 稳定基座之上，为系统引入多引擎支持能力和统一引擎信息架构。从单一本地引擎扩展到支持 HTTP 引擎、多版本引擎，建立从配置层到执行层再到展示层的完整引擎信息链路。

**核心成果**：
- S3-A：HTTP 引擎真实接入（已封板）
- S3-B：引擎配置与版本管理体系（已完成）
- S3-C：Studio/导入页联动增强（Phase 1 已完成）
